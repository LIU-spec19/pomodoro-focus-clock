## 番茄专注时钟（Pomodoro Timer）

一个基于番茄工作法（Pomodoro Technique）的专注计时应用，帮助你高效规划专注时间、休息时间，并记录你的专注历史与任务完成情况。

---

### 功能总览

- **番茄计时核心功能**
  - 可在 **专注 / 短休息 / 长休息** 三种模式间切换
  - 支持自定义每种模式的时长（如 25/5/15 分钟等）
  - 支持开始、暂停、重置计时
  - 计时结束时提供明显提醒（声音 / 视觉高亮）

- **任务管理（待办列表）**
  - 添加、编辑、删除任务
  - 为任务设置预期番茄数
  - 标记任务完成状态
  - 自动统计每个任务实际完成的番茄数

- **统计与可视化**
  - 展示每日 / 每周完成的番茄数量
  - 以图表或数字形式显示专注时间总和
  - 统计专注 / 短休息 / 长休息的使用比例

- **日历视图**
  - 查看某天的专注记录与完成番茄数
  - 快速对比一段时间内的专注趋势

- **提醒与交互体验**
  - 计时结束时的声音提醒（可开关）
  - 可选浏览器通知（如果用户授权）
  - 动效与视觉反馈，避免“忘记在计时”

- **数据持久化**
  - 使用本地存储（localStorage / IndexedDB 等）保存：
    - 任务列表
    - 统计数据
    - 用户配置（时长设置、声音开关等）
  - 即便刷新页面或重启浏览器，数据仍会保留

### 使用说明

- **开始一次番茄**
  - 在主界面选择模式（专注 / 短休息 / 长休息）
  - 点击“开始”按钮，计时开始
  - 再次点击可暂停；如需从头开始，可点击“重置”

- **管理任务**
  - 在任务区域输入任务名称，点击“添加任务”
  - 可以为任务设置目标番茄数（例如：预期需要 4 个番茄）
  - 每当一次专注番茄结束，自动为当前选中任务累加 1
  - 任务完成后可点击“完成”或删除

- **查看统计**
  - 打开统计页面 / 区域，可查看：
    - 当天完成番茄数
    - 最近一周趋势图
    - 总专注时间等信息

- **日历视图**
  - 在日历中点击具体日期，可查看当天的番茄完成情况
  - 适合回顾一段时间的专注习惯

### 配置与个性化

（以下为推荐示例，具体以实际实现为准）

- **时间设置**
  - 专注时长：默认 25 分钟，可在设置中调整

### 常见问题（FAQ）

- **Q：为什么我刷新页面后任务和统计没有丢失？**  
  **A：** 应用会把任务、配置和部分统计数据保存到浏览器本地存储中。

- **Q：可以在没有网络的情况下使用吗？**  
  **A：** 可以，本项目是纯前端应用，只要浏览器能打开本地或已部署的页面，就能离线使用（但不包含 PWA 安装特性，除非额外配置）。

- **Q：如何重置所有数据？**  
  **A：** 目前推荐方式是清除浏览器对该站点的存储数据（localStorage）；如后续增加“重置数据”按钮，以页面说明为准。
